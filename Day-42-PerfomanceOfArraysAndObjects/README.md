<h1>Analysing the performance of Objects and Arrays<h1>

- We'll look at the perfomance of arrays and objects.

<h2>Big O of Objects</h2>

- Insertion - O(1)
- Removal - O(1)
- Searching - O(N)
- Access - O(1)

<h2>Big O of Object Methods</h2>

- Object.keys - O(N)
- Object.values - O(N)
- Object.entries - O(N)
- hasOwnProperty - O(1)
